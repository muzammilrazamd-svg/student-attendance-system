#  Student Attendance System

A Full Stack Student Attendance Management System built using React, Express.js and MySQL.

## Features
- Add Student
- Search Student
- Edit Student(rename)
- Delete Student
- Mark Present
- Mark Absent
- Attendance Percentage
- Export CSV
- Dark Mode

## Tech Stack
- React
- Express.js
- MySQL
